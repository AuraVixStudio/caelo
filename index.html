Five modes, one hub

Everything xAI, in one place

Caelo unifies the full Grok surface behind a single shared backbone — context, history, projects and cost flow across every mode.

💬

Chat

Responses API with live web & X search, vision, document Q&A, citations and a live cost counter.

🎨

Image & Video

Generate and edit images and video through one async job queue, with a gallery and per-job cost.

🎙️

Voice

Text-to-speech, dictation, a real-time “Live” agent and a “Talk” pipeline — the key never reaches the UI.

⌨️

Coding agent

Sandboxed file tools, diff approval, trust modes, checkpoints/undo, project rules, and subagent teams.

🧩

Extensible

MCP servers, slash commands, hooks, skills, a package marketplace, headless CLI, ACP and LSP.

🗂️

History & projects

A searchable SQLite backbone records every mode; send artifacts between modes and scope work to projects.

Local-first by design

Your keys, your machine

● Bring your own key

You supply your own xAI credentials (OAuth or API key). They stay on your machine and are sent only to api.x.ai.

● No telemetry

Caelo collects nothing. The backend binds to loopback only and is never exposed on a routable interface.

● Open source

Apache-2.0 licensed. The mature xAI logic is reused, not rewritten — auditable end to end.

Ready to bring your own sky?

Download the Windows installer, sign in with your xAI account or paste an API key, and start.
