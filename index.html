Everything xAI, in one place
Five modes. One shared backbone.

Context, history, projects and cost flow across every mode — no more juggling five separate tools.

Chat

Live answers, with receipts.

Streaming chat with live web & X search, vision and document Q&A — every claim backed by clickable citations, with a running token & cost counter.

  • ✓ Web & X search with a Sources panel
  • ✓ Vision & document Q&A
  • ✓ Real per-request cost & token counter
Code

An agent you can actually trust.

A mini-IDE with a real coding agent: sandboxed file tools, unified diff approval, checkpoints & undo, plan mode, project rules — and parallel subagent teams.

  • ✓ Diff approval & 4 trust modes
  • ✓ Checkpoints, undo & project rules (CAELO.md)
  • ✓ Subagent teams with merge review
Image & Video

Create, edit, and keep it all.

Generate and edit images and video through one async job queue, with variations, edit/extend, and a unified searchable gallery scoped to your projects.

  • ✓ Text→image/video, edit & variations
  • ✓ Server-side video polling — never blocks
  • ✓ Unified gallery & per-job cost
Voice

Talk to Grok, hands-free.

Text-to-speech replies, dictation, a full voice conversation pipeline, and a low-latency realtime “Live” mode — your key never reaches the browser.

  • ✓ Speak (TTS) & Transcribe (STT)
  • ✓ Talk pipeline & realtime Live
  • ✓ Per-session audio cost
Programmable
Built to extend.

MCP servers (local & remote), slash commands, hooks, skills, a package marketplace, headless CLI, plus ACP & LSP.

Local-first by design
Your keys. Your machine.
🔑

Bring your own key

Sign in with your xAI account or paste an API key. It stays on your machine and is sent only to api.x.ai — never returned by the local API.

🛡️

Zero telemetry

No analytics, no usage reporting. The backend binds to loopback only and every request is token-authenticated and fail-closed.

📂

Open & auditable

Apache-2.0 licensed. The agent is sandboxed to your workspace; writes and shell commands require approval, with a secret-free environment.

Bring your own sky.

Download the signed Windows installer, sign in with your xAI account or paste an API key, and start.
